# {{ page.meta.title }}

## :octicons-square-fill-16: 函数说明

将 **东亚模糊宽度字符** 视为宽度为 1 的窄字符

本函数影响 **Unicode 标准** 的所有版本

本函数影响 [CW_CharWidth](CW_CharWidth.md) 对 **东亚模糊宽度字符** 宽度的判断

## :octicons-square-fill-16: 函数原型

```C
void CW_TreatAmbiguousAsNarrow();
```
